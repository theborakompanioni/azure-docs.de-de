# <a name="discover-and-share-tutorials-in-cortana-intelligence-gallery"></a>Entdecken und Freigeben von Tutorials im Cortana Intelligence-Katalog
[!INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="tutorials"></a>Tutorials
Der Cortana Intelligence-Katalog bietet mehrere [Tutorials](https://gallery.cortanaintelligence.com/tutorials), die Ihnen die Technologien und Konzepte für maschinelles Lernen näherbringen. In den Tutorials können auch fortgeschrittene Methoden für Probleme im Zusammenhang mit maschinellem Lernen beschrieben sein.

## <a name="discover"></a>Entdecken
Zum Durchsuchen der Tutorials im [Katalog](http://gallery.cortanaintelligence.com) wählen Sie unter **Mehr** die Option **Tutorials** aus.

![Wählen Sie die Tutorials auf der Startseite des Katalogs aus](media/machine-learning-gallery-tutorials/select-tutorials-in-gallery.png)

Auf der Seite **[Tutorials](https://gallery.cortanaintelligence.com/tutorials)** finden Sie eine Liste der zuletzt hinzugefügten und beliebtesten Tutorials. Um alle Tutorials zu sehen, wählen Sie die Schaltfläche **Alle anzeigen** aus. Wählen Sie zum Suchen nach einem bestimmten Tutorial **Alle anzeigen** und dann die Filterkriterien aus. Sie können auch Suchbegriffe im Feld **Suchen** im oberen Bereich der Seite „Katalog“ eingeben.

Weitere Informationen zu einem Tutorial erhalten Sie auf der Detailseite des Tutorials. Um die Detailseite eines Tutorials zu öffnen, wählen Sie das Tutorial aus. Auf der Detailseite eines Tutorials können Sie im Abschnitt **Kommentare** kommentieren, Feedback abgeben oder Fragen zum Tutorial stellen. Sie können das Tutorial sogar für Ihre Kontakte oder Kollegen auf Twitter oder LinkedIn freigeben. Sie können auch einen Link zur Detailseite für Tutorials per E-Mail senden, um andere Benutzer zum Besuch der Seite einzuladen.

![Teilen Sie dieses Element mit Freunden](media/machine-learning-gallery-how-to-use-contribute-publish/share-links.png)

![Fügen Sie Ihre eigenen Kommentare hinzu](media/machine-learning-gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>Mitwirken
Sie können ein Tutorial dem Katalog hinzufügen, um anderen Benutzern dabei zu helfen, ein Problem zu lösen oder mehr über ein Konzept zu erfahren.

### <a name="create-a-tutorial"></a>Erstellen eines Tutorials

1. Melden Sie sich mit Ihrem Microsoft-Konto beim Katalog an.

2. Wählen Sie Ihr Bild oben auf der Seite und dann Ihren Namen aus.
  
    ![Wählen Sie den Namen Ihres Kontos aus](media/machine-learning-gallery-tutorials/click-account-name.png)

3. Wählen Sie **Neues Element** aus.
  
    ![Auswählen von „Neues Element“](media/machine-learning-gallery-collections/click-new-item.png)

4. Wählen Sie auf der Seite **Beschreibung** für **ELEMENTTYP** die Option **Tutorial** aus. Geben Sie einen Namen für das Tutorial, eine kurze Zusammenfassung, eine ausführliche Beschreibung und beliebige Tags ein, mit denen andere Benutzer Ihr Tutorial finden können.
  
    ![Eingeben von Informationen für das neue Tutorial](media/machine-learning-gallery-tutorials/create-tutorial-page-1.png)
5. Wählen Sie auf der Seite **Bildauswahl** ein Bild aus, das mit der Sammlung angezeigt wird. Sie können eine eigene Bilddatei hochladen oder ein vorgefertigtes Bild auswählen. Wählen Sie ein Bild aus, durch das die Benutzer Inhalt und Zweck des Tutorials erkennen können.
  
    ![Eingeben von Informationen für das neue Tutorial](media/machine-learning-gallery-tutorials/create-tutorial-page-2.png)

6. Wählen Sie auf der Seite **Einstellungen** für **Sichtbarkeit** aus, ob die Sammlung **Öffentlich** (jeder kann das Tutorial anzeigen) oder **Nicht aufgeführt** ist (nur Personen mit einem direkten Link zum Tutorial können es anzeigen).
  
    > [!IMPORTANT]
    > Nach dem Festlegen eines Tutorials auf **Öffentlich** und der Auswahl von **Erstellen** können Sie dieses Tutorial später nicht mehr in **Nicht aufgeführt** ändern.
    > 
    > 
  
    ![Wählen Sie „Öffentlich“ oder „Nicht aufgeführt“ aus.](media/machine-learning-gallery-tutorials/create-tutorial-page-3.png)

7. Klicken Sie auf **Erstellen**.

Ihr Tutorial befindet sich jetzt im Cortana Intelligence-Katalog. Ihre Tutorials werden auf der Seite Ihres Kontos unter der Registerkarte **Elemente** angezeigt.

**[Wechseln zum Katalog](http://gallery.cortanaintelligence.com)**
