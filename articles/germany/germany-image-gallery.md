# <a name="azure-germany-images"></a>Azure Deutschland-Images
## <a name="overview"></a>Übersicht
Kunden können beim Bereitstellen neuer virtueller Server in Azure Deutschland wahlweise vorgefertigte Images von Microsoft verwenden oder ihre eigenen VHDs hochladen. So erhalten Sie die Flexibilität, bei Bedarf Ihre eigenen standardisierten Images verwenden zu können.

## <a name="variations"></a>Abweichungen
Images unserer Partner werden gegenwärtig noch nicht nativ im Azure Deutschland-Marketplace angeboten.

Nachfolgend finden Sie eine Liste verfügbarer Images. Beachten Sie bitte, dass diese möglicherweise in neueren Versionen vorliegen können, die hier noch nicht aufgelistet sind. Für einige Images gelten separate Lizenzierungs- und Bezahloptionen. Besuchen Sie die Seite [Azure-Preise](https://azure.microsoft.com/pricing/), um weitere Informationen zu erhalten, oder wenden Sie sich an Ihren Microsoft-Kundenberater oder Ihren Partner, um die für Sie geeignetsten Optionen zu ermitteln.
