### <a name="prerequisites"></a>Voraussetzungen
* Ein [Salesforce](https://salesforce.com)-Konto  

Bevor Sie Ihr Salesforce-Konto in einer Logik-App verwenden können, müssen Sie die Logik-App für die Verbindung mit Ihrem Salesforce-Konto autorisieren. Glücklicherweise können Sie dies ganz einfach in Ihrer Logik-App im Azure-Portal durchführen.  

Hier finden Sie die Schritte zum Autorisieren Ihrer Logik-App für Verbindungen mit Ihrem Salesforce-Konto:  

1. Um eine Verbindung mit Salesforce herzustellen, wählen Sie im Logik-App-Designer in der Dropdownliste **Von Microsoft verwaltete APIs anzeigen** aus, und geben Sie dann *Salesforce* in das Suchfeld ein. Wählen Sie den zu verwendenden Trigger oder die gewünschte Aktion aus:   
   ![Salesforce-Verbindung – Abbildung 1](./media/connectors-create-api-salesforce/salesforce-1.png)  
2. Wenn Sie noch keine Verbindungen mit Salesforce hergestellt haben, werden Sie aufgefordert, Ihre Salesforce-Anmeldeinformationen anzugeben. Mit diesen Anmeldeinformationen wird Ihre Logik-App autorisiert, eine Verbindung mit dem Salesforce-Konto herzustellen und auf dessen Daten zuzugreifen:  
   ![Salesforce-Verbindung – Abbildung 2](./media/connectors-create-api-salesforce/salesforce-2.png)  
3. Geben Sie Ihren Salesforce-Benutzernamen und das zugehörige Kennwort zum Autorisieren Ihrer Logik-App an:  
   ![Salesforce-Verbindung – Abbildung 3](./media/connectors-create-api-salesforce/salesforce-3.png)  
4. Erlauben Sie die Verbindung mit Salesforce:  
   ![Salesforce-Verbindung – Abbildung 4](./media/connectors-create-api-salesforce/salesforce-4.png)  
5. Beachten Sie, dass die Verbindung erstellt wurde und Sie nun mit den anderen Schritten in Ihrer Logik-App fortfahren können:  
   ![Salesforce-Verbindung – Abbildung 5](./media/connectors-create-api-salesforce/salesforce-5.png)
